# 🎂 Happy Birthday 💖

> Live site: [pallab-chakraborty.github.io/Her-Birthday](https://pallab-chakraborty.github.io/Her-Birthday/)

A heartfelt, interactive birthday surprise website — personally designed and built as a special gift. A unique blend of creativity and code to celebrate someone special in the most memorable way.

---

## 💕 About the Project

This is a custom-built birthday experience website made with love for **Srija**. Instead of a card or a message, this project turns a birthday wish into a full interactive web experience — complete with animations, surprises, and personal touches.

> *"Click to Enter Our World 💕"*

---

## ✨ Features

- 🎊 Animated birthday landing page with festive visuals
- 💖 Personal and intimate design made for one person
- 🎉 Interactive entrance — click to enter the experience
- 🎈 Smooth animations and celebratory effects
- 📱 Fully responsive — looks beautiful on mobile and desktop
- 🌸 Custom theme built around a personal aesthetic
